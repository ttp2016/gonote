### 1.Ctrl+s后自动格式化代码
> File --> Settings Tools --> File Watchers  --> +  --> go fmt

### 2.快捷键设置
> File --> Setting  --> KeyMap

### 3.goland 文件头注释
打开GoLand的setting选项  
依次选择Editor，CodeStyle ，File and Code Templates ,Go File 

根据自己需要添加即可

```
/*
/*
@Time : ${DATE} ${TIME} 
@Author : ${USER}
@File : ${NAME}
@Software: ${PRODUCT_NAME}
*/
package ${GO_PACKAGE_NAME}
*/
```
